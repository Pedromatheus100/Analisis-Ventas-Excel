# Analisis-Ventas-Excel
Proyecto de Analisis de Datos en Excel que incluye Base de Datos Estructurada, tablas dinamicas, graficos interactivos y segmentadores de datos para el reporte de indicadores de ventas

#Este proyecto demuestra habilidades de procesamiento y visualización de datos utilizando Microsoft Excel.

Funcionalidades principales:
- Base de Datos: Estructuración de datos transaccionales con campos de fecha, vendedor, producto y montos.

- Tablas Dinámicas: Agrupación y resumen de información para el análisis temporal de ventas.

- Interactividad: Implementación de segmentadores de datos (slicers) para filtrar información por vendedor y producto de forma dinámica.

- Visualización: Gráficos de ventas que permiten identificar rápidamente el rendimiento de los productos y vendedores.

- Proyecto desarrollado para la gestión y análisis eficiente de información comercial.
